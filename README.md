# 仿小红书界面的读书分享系统
该系统是 java Spring 的 课设，只是一个demo 


# 部署
- jdk 17
- mysql 8.0
- redis 3.5
- tomcat 9
- 需要在环境变量/代码中配置邮箱相关秘钥
  - utils.EmailUtil 中配置邮箱服务器地址
  - utils.EmailUtil 中配置邮箱服务器端口
  - 环境变量 EmailAccount 中配置邮箱账号
  - 环境变量 EmailSecret 中配置邮箱账号秘钥
